# 历史归档（Archive）

本目录仅承载“阶段交付/一次性审计/历史方案/旧任务记录”，避免与现行 Wiki 混在一起。

## 目录
- `_audit/`：2025-10 项目审计与重组材料
- `dashboard-refactor/`：Dashboard 重构阶段交付/交接材料
- `jwt改/`、`jwt改造/`：JWT 改造历史材料
- `task/`：历史任务交付文档
- `handovers/`：零散交接/摘要（按需引用）

## 规则（KISS）
1. **新文档默认写入现行目录**（不要直接写进 archive）。
2. **归档只做“移动 + 导航修正”**，不改写历史结论；如需补充现状，追加“现状说明”小节。
