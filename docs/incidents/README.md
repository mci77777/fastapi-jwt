### 查找事件记录
```bash

# 按严重程度搜索
grep -r "🔴 高危" docs/incidents/

```

### 添加新事件记录
```bash
# 命名格式：YYYY-MM-<事件简述>.md
# 示例：2024-11-ddos-attack.md

# 模板结构：

# 3. 根因分析
# 4. 应急措施
# 5. 长期改进
# 6. 经验教训
```

## 📊 事件分类

### 严重程度定义
- 🔴 **高危**：数据泄露、服务中断、安全漏洞
- 🟡 **中等**：性能下降、功能异常、配置错误
- 🟢 **低危**：日志异常、监控告警、非关键功能

### 事件类型
- **安全事件**：密钥泄露、未授权访问、DDoS 攻击
- **运维事件**：服务中断、数据库故障、部署失败
- **迁移事件**：仓库迁移、数据迁移、架构升级
